# Today I Learned

본 디렉토리는 학습한 내용이나 성장하는 과정에서 경험한 시행착오들을 기록해 놓은 공간입니다.  

꾸준히 성장하기 위해 한 번 기록하고 끝나는 방식의 학습이 아닌  
성장 과정에서 얻은 경험을 바탕으로 퇴고를 거듭하며 새로운 깨달음을 얻고 견문이 넓어지는 학습을 지향합니다.  
  
따라서 기록된 내용은 항상 미완이며 언제든지 수정, 변경될 수 있습니다.

README 형식은 [namjunemy](https://github.com/namjunemy)님의 Github TIL를 참조하였습니다.

---
## 규칙

- 원작자가 허용하는 자료만 사용하고 내용의 Reference를 명시한다.
- 진정한 학습은 퇴고에서 비롯되며 기록을 방치하지 않는다.
