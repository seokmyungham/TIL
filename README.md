# Today I Learned

본 디렉토리는 학습한 내용이나 성장하는 과정에서 경험한 시행착오들을 기록해 놓은 공간입니다.  

꾸준히 성장하기 위해 한 번 기록하고 끝나는 방식의 학습이 아닌  
성장 과정에서 얻은 경험을 바탕으로 기록을 다시 수정해 보면서, 새로운 깨달음을 얻고 견문이 넓어지는 학습을 지향합니다.  

따라서 기록된 내용은 언제든지 수정, 변경될 수 있습니다.

README 형식은 [namjunemy](https://github.com/namjunemy/TIL/tree/master)님의 Github를 보며 참고하였습니다.
